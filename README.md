# Ochem Visualizer

This repo has been cleaned up to keep one canonical Next.js setup while you focus on the organic chemistry mechanism visualizer direction.

## Near-term direction

- Primary focus: organic chemistry mechanisms + visualizer experience.
- Planned addition: a dedicated **Biology** tab so the biological angle can grow without taking over the core ochem flow.

## Current setup

- Next.js 15 + TypeScript + Tailwind
- Single canonical Next config: `next.config.ts`
- Build artifacts are git-ignored (for example `*.tsbuildinfo`)

## Getting started

```bash
npm install
npm run dev
```
